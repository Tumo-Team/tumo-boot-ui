## Introduction

基于 [vben-admin-thin-next](https://github.com/anncwb/vben-admin-thin-next)

## Install and use

- Get the project code

```bash
git clone https://github.com/anncwb/vue-vben-admin.git
```

- Installation dependencies

```bash
cd vue-vben-admin

yarn install

```

- run

```bash
yarn serve
```

- build

```bash
yarn build
```
